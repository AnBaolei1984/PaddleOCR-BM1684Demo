# PaddleOCR-CppDemo
这个是在比特大陆BM1684系列AI硬件上支持PaddleOCR的demo,可以支持x86(SC5)和Arm(SE5/SM5)两类设备。

## 编译

- 编译x86执行 sh tools/build_x86.sh
- 编译arm执行 sh tools/build_arm.sh

## 运行

sh tools/run.sh
